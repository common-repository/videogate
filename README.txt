== Description ==

VideoGate is a user-friendly WordPress video plugin to showcase your videos. You can embed both youtube and vimeo videos.


== Installation ==

Itâ€™s pretty easy to set up videogate plugin by following steps below.

1. Upload the plugin files to the `/wp-content/plugins/videogate` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. go to videogate -> click add new video and publish your video.
1. use [videogate_videos column="2"] to display video on any page, second parameter is to set column by default is 2 column while you can set up to 6 columns.
